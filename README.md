# REGISTER

##  Objetivo Geral
Facilitar o cadastro de clientes para empresas de impermeabilização de estofados, automatizando o envio de informações das lojas para a empresa principal.

##  Principais Funcionalidades
- Cadastrar dados de clientes (nome, endereço, telefone);
- Registrar informações sobre o móvel e o valor da impermeabilização;
- Calcular o total e aplicar desconto;
- Enviar automaticamente os cadastros para a empresa;
- Permitir consulta e gerenciamento dos cadastros.
